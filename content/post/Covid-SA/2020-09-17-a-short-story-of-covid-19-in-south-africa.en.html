Introduction

This blog will analyse the COVID-19 cases of South Africa. Relative to the previous blog of COVID-19 cases in Singapore South Africa’s economic status, infrastructure, education levels, are generally inferior to that of Singapore as they are a third world country. Furthermore, South Africa’s population is approximately ten folds of that of Singapore.

In examining the COVID-19 cases in South Africa, I will break down the cases into distinct provinces of South Africa to verify if a bigger population is associated with a larger number of confirmed cases, deaths and recovered cases.

The table above shows the top 100 countries with the highest confirmed cases. As of today, South Africa ranked 6th in confirmed cases in the world.

Getting the Data

To analyse the COVID-19 cases, I extracted the daily summary of COVID-19 cases in the coronavirus package. This package contains the raw data in a tidy format, summarising the daily cases by state/province and is readily available in this GitHub Link. Each observation represents the total number of cases recorded in the state/province, where each cases are identified by their case type, spanning from 2020-01-23 and it updated regularly. It was really convenient to use this data set as no data wrangling had to be done.

Next, I downloaded the population data and cumulative confirmed, deaths and recovered cases from this GitHub Link (@marivate2020use). The cleaning process included using regular expressions to manipulate character strings, and relational joins to join data sets with matching observations. The cleaned data set was in the same format with that of the coronavirus dataset.
